## 下载

### Docker

```
docker pull apachecn0/apachecn-js-zh
docker run -tid -p <port>:80 apachecn0/apachecn-js-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install apachecn-js-zh
apachecn-js-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-js-zh
apachecn-js-zh <port>
# 访问 http://localhost:{port} 查看文档
```
